## Question 1
Suppose I first execute the following Octave/Matlab commands:
```
A = [1 2; 3 4; 5 6];
B = [1 2 3; 4 5 6];
```
Which of the following are then valid commands? Check all that apply. (Hint: A' denotes the transpose of A.)
### Answer
* C = A * B;
* C = B' + A;
